clearTimeout

(
  • id
)
final static

Defined in src/o2/timer/core.js:215

Clears the timer scheduled with the given id.

Parameters:

  • id Number
    • the id of the timer.

Example:

var timer = require('amd/o2/timer/core');

var id = timer.setTimeout(function() {
    console.log('This will run at least after a second');
}, 1000);

...

// Now the task won't run.
timer.clearTimeout(id);

Defers tasks to requestAnimationFrame.

Use this instead of window.setTimeout.

Parameters:

  • delegate Function
    • the delegate to execute in the future.
  • timeout Number
    • timeout in milliseconds.

Returns:

Number:
  • a timeout id that we can use to clear the timeout.
